HTML5 E CSS3

Desde a sua implementação, no início dos anos 90, o HTML passou por
diversas melhorias tornando-se a base para blogs, plataformas de
e-commerce, redes sociais e todo tipo de página que se pode acessar
através de um navegador web. Em 2014, o W3C (The World Wide Web
Consortium) publicou a quinta revisão do padrão HTML, que está se
atualizando até hoje. As mudanças visam facilitar o trabalho dos
desenvolvedores web, além de melhorar a experiência do usuário.

O HTML5 é a última versão dA Linguagem de Marcação de Hipertext.
Combina três tecnologias web: HTML, para construir a estrutura
da página; CSS, para melhorar a apresentação; e JavaScript, para
adicionar funcionalidade aos elementos HTML. Suas tags ou marcas
fazem com que textos, imagens e elementos multimídia sejam
visualizados corretamente na tela, bem como permitem que as
páginas web tenham um formato unificado, padronizado e sejam
compatíveis com diferentes navegadores como Chrome, Edge,
Firefox e Safari.

Quais são as novidades?

Tags ou elementos semânticos ( header, nav, section,
article, footer, etc.).Tags de multimídia (audio, video).
Novas formas de controle como atributos, tipos de entrada,
barra de progresso, etc.A tag canvas para desenhos
2D usando JavaScript.A tag svg para desenhos baseados em
vetores, formas, gráficosde pizza e de barras, mapas,
etc.A geolocalização para descobrir a localização do usuário.
